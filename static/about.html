About No-Frills API

No-Frills API is designed to make getting information about No-Frills grocery stores easy and smooth. It retrieves and saves info about all store locations - everything from address and manager to departments, as well as current flyers and all the product information therein. This data is then accessible at various endpoints, in various combinations.

Please let us know if there's anything you'd like to see added to this API - or better yet, feel free to contribute on Github!

Enjoy your on-sale avocadoes responsibly!
